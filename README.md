# 💰 AI Budget Advisor

A web app that analyzes your monthly budget and gives you personalized financial advice powered by Claude AI.

## What it does

- Enter your monthly income
- Add your expenses by category
- Click **Analyze My Budget** to get instant AI-powered budgeting advice

## Built with

- HTML & CSS — structure and styling
- JavaScript — interactivity and logic
- Claude AI (Anthropic API) — budget analysis and advice

## How to run it

1. Clone or download this repo
2. Add your Anthropic API key to `app.js`
3. Open `index.html` in a browser
4. Enter your income and expenses and click Analyze!

## What I learned

- How to build a frontend web app from scratch
- How to connect to and use an AI API
- How to handle asynchronous JavaScript with async/await
- How to dynamically update a webpage with JavaScript
